# SageMaker
SageMakerは、AIを使用するための環境を作成するサービスです。データをアップロードして、AIを使用することができます。
## SageMaker Studio
WebのUI上でAIを使用するための環境を作成する総合開発環境
## SageMaker Neo
機械学習モデルをクラウドやエッジデバイスで高速・効率的に動作させるためのサービス
## Amazon SageMaker Serverless Inference

## SageMaker JumpStart
事前学習モデルを簡単に微調整、デプロイすることができるサービス
# Amazon SageMakerAIに組み込めるアルゴリズム
## 教師なし
### Random Cut Forest 
時系列データから非典型的なデータを検出し異常をリアルタイムに検知するアルゴリズム
### DeepAR
時系列データから未来の値を予測するアルゴリズム
### IP Insights
IPv4アドレスの仕様パターンを学習する教師なしアルゴリズム異常なIPアドレスからのウェブサービスの攻撃を検知する。
